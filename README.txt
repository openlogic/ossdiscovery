Following are the possible ways a person would be interested in checking out 
this project's source code depending on what they're interested in doing.

* If you only want to work with the content of the project site, check out the 
  'www' directory.

* If you only want the main source responsible for making this application go at 
  runtime, check out the 'app/main' directory.  For more detailed, 
  developer-oriented information, see the README.txt file in 'app/main'.

* If you want the previously mentioned main runtime source, plus all the tests 
  and test resources, check out the 'app' directory.

* If you want everything, simply check out the trunk.
