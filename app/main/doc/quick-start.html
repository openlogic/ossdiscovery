Step 3: Run OSS Discovery On One Machine and Automatically Contribute Scans

Important Information

Before you get started, please take note of these important points:

  • You can only submit one scan from the same machine per 24 hour period.
  • Scans can take several minutes to several hours depending on the size and content of your machine. Most desktop machines scan in under an hour. You can continue to use your machine while the scan is running.
  • Scans on Linux/Unix will include all mounted shares.
  • You may encounter an error if your company uses a proxy or authenticated server.
  • FreeBSD users must, as root, create a symlink to access bash using the following command: ln -s /usr/local/bin/bash /bin/bash

For Linux & Solaris

In a shell, CD to the directory where you extracted OSS Discovery and execute:

./discovery --census-code YOUR-CENSUS-CODE-GOES-HERE --deliver-results

Replace YOUR-CENSUS-CODE-GOES-HERE with the Census Code you received at registration. Right-click in the command window to paste text.

Note: The Census Code is case sensitive and must be used exactly as provided at registration, including dashes. You must include the Census Code in order to see your reports on the Census website. You can retrieve your Census Code at any time by logging into the Census website and clicking the "My Account" link in the upper-right corner.

For Windows

At the command line, CD to the directory where you extracted OSS Discovery and run:discovery.bat --census-code YOUR-CENSUS-CODE-GOES-HERE --deliver-results

Replace YOUR-CENSUS-CODE-GOES-HERE with the Census Code you received at registration. Right-click in the command window to paste text.

Note: The Census Code is case sensitive and must be used exactly as provided at registration, including dashes. You must include the Census Code in order to see your reports on the Census website. You can retrieve your Census Code at any time by logging into the Census website and clicking the "My Account" link in the upper-right corner.
