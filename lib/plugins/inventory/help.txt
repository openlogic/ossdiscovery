
Inventory Plugin v1.0 --help
--group-code,        -C the identifier code received from OpenLogic registration time
--inventory-local,   -u the absolute or relative path and filename for the human readable results file.  The default is STDOUT
--inventory-results, -m the absolute or relative path and filename for the machine readable results file to submit to OpenLogic
--list-plugins,      -N lists just the enabled plugins, not all plugins installed
